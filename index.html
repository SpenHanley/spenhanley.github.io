Projects

Turtle Thingy

This was developed as a simple side project to allow me to practice using and drawing to BufferedImages in the java language. It is based on a toy that was brought into school when I was younger, where you would provide it with instructions and it would follow them. It was also possible to add a pen to the base of the Turtle and have it draw something given the correct commands.

This project is still being developed as I intend on adding support for a simple custom scripting language to allow for more complex control of the turtle. This project is also going to be used as part of a program for young people in scout groups to allow them to learn something about programming in a simple way that provides instant feedback to their commands.
